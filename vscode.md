## Code Tricks

#####  Using Ctrl keys
```
Ctrl + f : Find
Ctrl + h : Replace
Ctrl + p : Go to File
Ctrl + g : Go to Line
```

## Editor Features

##### Multi-line edit
```
Ctrl + Alt + Up/Down Arrow : Use cursor on multiple lines simultaneously
```
